ClipFlow

ClipFlow is a web-based video processing tool that allows users to create short clips from long videos and upload them to their TikTok accounts using the official TikTok Content Posting API.

Features

  • Video processing and clip creation tools
  • Support for multiple video sources
  • Formatting videos for vertical viewing (9:16)
  • Upload to TikTok using the official API

Privacy Policy

Last updated: March 2026

ClipFlow ("the app") is committed to protecting your privacy. This Privacy Policy explains how ClipFlow handles information.

Data Collection: ClipFlow does not collect, store, or share personal data on external servers. The application operates locally on the user's device.

TikTok API Usage: ClipFlow uses TikTok's official Content Posting API to upload video content on behalf of authenticated users. Authentication is handled via TikTok OAuth. Access tokens are stored locally on the user's device and are only used to communicate with TikTok's official API.

No Third-Party Sharing: ClipFlow does not sell, trade, or share user data with third parties.

User Control: Users maintain full control over the content they process and upload.
