ClipBot

ClipBot is a video clip automation tool that helps content creators automatically extract short clips from long videos and publish them to TikTok using the official TikTok Content Posting API.

Features

  • Automatic video downloading from YouTube and other platforms
  • Smart clip extraction using audio analysis
  • Automatic formatting to TikTok vertical format (9:16)
  • Direct publishing to TikTok via official API

Privacy Policy

Last updated: March 2026

ClipBot ("the app") is committed to protecting your privacy. This Privacy Policy explains how ClipBot handles information.

Data Collection: ClipBot does not collect, store, or share any personal data on external servers. The app runs locally on the user's device.

TikTok API Usage: ClipBot uses TikTok's official Content Posting API solely to upload video clips on behalf of the authenticated user. The TikTok access token is stored locally on the user's device and is never transmitted to any third party other than TikTok's official API endpoints.

No Third-Party Sharing: ClipBot does not sell, trade, or share any user data with third parties.
